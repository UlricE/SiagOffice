The Toolbar

The toolbar uses icons to simplify some of the most common operations.
Starts a new instance of Siag.
Open a file.
Save the document.
Preview the document prior to printing (requires Ghostview).
Print the document.
Plots a graph (requires Gnuplot).
Inserts the sum of all the cells in the block into the current cell.
Displays help for Siag (requires Chimera).
Displays the Gnu General Public License.
Draws a border around the block.
Draws grid lines in the block.
Underlines the block.
Removes all grid lines from the block.
Toggles bold text.
Toggles italic text.
Left adjust text.
Center text.
Right adjust text.
