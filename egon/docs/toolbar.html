The Toolbar

The toolbar uses icons to simplify some of the most common operations.
Starts a new instance of Egon Animator.
Open a file.
Save the document.
Preview the document prior to printing (requires Ghostview).
Print the document.
Pauses the animation.
Single-steps the animation one step backward.
Starts the animation.
Single-steps the animation one step forward.
Displays help for Egon Animator (requires Chimera or another HTML browser).
Displays the Gnu General Public License.
